# 熊戰擂台
**隱私策略**

本隱私政策適用於由深圳社源科技有限公司（以下簡稱「服務提供商」）作為免費增值服務創建的移動設備版應用程序（以下簡稱「應用程序」）。此服務旨在「按原樣」使用。

**信息收集和使用**

應用程序會在您下載和使用時收集信息。此信息可能包括以下信息

*   您設備的互聯網協議地址（例如 IP 地址）
*   您訪問的應用程序頁面、訪問的時間和日期、在這些頁面上花費的時間
*   在應用程序上花費的時間
*   您在移動設備上使用的操作系統

The Application does not gather precise information about the location of your mobile device.

該應用程序會收集您設備的位置，這有助於服務提供商確定您的大致地理位置，並通過以下方式使用：

*   地理定位服務：服務提供商利用位置數據提供個性化內容、相關推薦和基於位置的服務等功能。
*   分析和改進：匯總和匿名的位置數據可幫助服務提供商分析用戶行為、識別趨勢並改進應用程序的整體性能和功能。
*   第三方服務：服務提供商可能會定期將匿名位置數據傳輸給外部服務。這些服務可幫助他們增強應用程序並優化其產品。

服務提供商可能會使用您提供的信息不時與您聯繫，以便為您提供重要信息、必要的通知和營銷促銷。

為獲得更好的體驗，在使用應用程序時，服務提供商可能會要求您向我們提供某些個人身份信息，包括但不限於個人身份識別件、用戶ID、年齡、性別信息等。服務提供商請求的信息將由他們保留，並按照本隱私政策中的說明使用。

**第三方訪問**

只有匯總的匿名數據會定期傳輸到外部服務，以幫助服務提供商改進應用程序及其服務。服務提供商可能會按照本隱私聲明中描述的方式與第三方共享您的信息。

服務提供商可能會在以下情況下披露用戶提供和自動收集的信息：

*   根據法律要求，例如遵守傳票或類似的法律程序;
*   當他們真誠地認為有必要披露以保護他們的權利、保護您或他人的安全、調查欺詐或響應政府要求時;

您可以通過卸載應用程序輕鬆停止應用程序的所有信息收集。您可以使用作為移動設備的一部分或通過移動應用程序市場或網絡提供的標準卸載流程。
