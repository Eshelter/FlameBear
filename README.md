<h1>隱私政策 – 熊戰擂台系列應用</h1>

<p>本隱私政策適用於以下應用程序：</p>
<ul>
<li>熊戰擂台-控制端</li>
<li>熊戰擂台-玩家端</li>
</ul>

<p>上述應用由深圳社源科技有限公司（以下簡稱「服務提供商」）提供，主要用於 AR 體育互動與校園活動場景。</p>

<h2>1. 應用角色說明</h2>
<ul>
<li><strong>控制端：</strong>主要用於教師或管理人員創建與管理對戰或訓練場景</li>
<li><strong>玩家端：</strong>用於參與 AR 對戰或訓練互動</li>
</ul>

<p>兩個應用可在同一局域網內協同運作，但功能與數據用途有所不同。</p>

<h2>2. 信息收集</h2>
<ul>
<li>設備信息：設備類型、操作系統、IP 地址、應用版本</li>
<li>局域網信息：用於設備之間建立連線（多人互動）</li>
<li>使用數據：應用使用時長、功能操作記錄</li>
<li>可選用戶信息：帳號、用戶ID（如學校或服務場景需要）</li>
</ul>

<p><strong>重要說明：</strong>控制端與玩家端之間的數據僅用於當前連線會話，不會用於建立跨設備個人檔案。</p>

<h2>3. 信息使用方式</h2>
<ul>
<li>建立設備之間的 AR 互動連線</li>
<li>支持控制端與玩家端的協同運作</li>
<li>改善應用穩定性與性能</li>
<li>提供學校或服務場景下的功能支持</li>
</ul>

<h2>4. AR 與設備權限</h2>
<p>應用可能需要以下權限：</p>
<ul>
<li>相機：用於 AR 畫面捕捉（僅本地處理）</li>
<li>本地網絡 / 藍牙：用於設備間連接</li>
</ul>

<p>所有影像與數據均在設備本地處理，不會存儲或上傳。</p>

<h2>5. 第三方服務</h2>
<p>我們可能使用分析工具（如 Unity Analytics）來提升應用性能。這些服務僅接收匿名數據。</p>

<h2>6. 兒童隱私</h2>
<p>本應用主要用於校園場景。我們不會在未經監護人同意的情況下收集 13 歲以下兒童的個人信息。</p>

<h2>7. 數據安全</h2>
<p>我們採取合理的技術與管理措施保護數據安全。</p>

<h2>8. 用戶權利</h2>
<p>您可以通過卸載應用停止數據收集。如需刪除數據，請聯繫我們。</p>
